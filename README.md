# Reels Booster — landing

Сверстано по дизайну в Figma (Landing page «Video Reels»):
- ПК: node `1492-21237`
- Мобильная: node `1536-21493`
- Контейнер «Программа практикума»: node `1539-23879`

## Структура файлов

```
landing/
├─ index.html      ← разметка всех 11 секций
├─ styles.css      ← все стили, mobile-first, 2 брейкпоинта
├─ scripts.js      ← табы программы, аккордеоны (через <details>), CTA-обработчик
├─ assets/         ← сюда положить картинки/иконки/видео (см. ниже)
└─ README.md       ← вы читаете
```

## Запуск

Откройте `index.html` в браузере (или поднимите статикой):
```bash
python3 -m http.server 8000   # http://localhost:8000
# или
npx serve .
```

## Адаптив

Mobile-first. Брейкпоинты:
| px        | что меняется                                                                 |
|-----------|------------------------------------------------------------------------------|
| < 768     | мобильная вёрстка (одноколоночная, hero-картинка снизу)                      |
| ≥ 768     | планшет: hero становится двухколоночным, шаги — зигзаг, тарифы — 3 в ряд     |
| ≥ 1280    | desktop: типографика по Figma (61px H1, 38px subtitle, 86px section title)   |
| ≥ 1600    | ultra-wide: контейнер 1440px, ещё больший hero                                |

Все размеры используют `clamp()`, поэтому переходы между брейкпоинтами плавные.

## Дизайн-токены

В `:root` лежат CSS-переменные с цветами и шрифтами из Figma:

```css
--c-bg:       #101010;   /* фон */
--c-text:     #f3f3f3;
--c-blue:     #5b5aff;   /* primary CTA */
--c-violet-100: #aa2b8e; /* акцент */
--c-pumpkin:  #ff9b47;
--c-scarlet:  #ff5f33;
--gradient-brand: linear-gradient(214.5deg, #ff831a 13%, #5b5aff 91%);
```

Шрифты: **Manrope** (близкий по форме к лицензионному *Azo Sans*) и **Inter**.
Если у вас есть лицензия Azo Sans — подключите его через `@font-face`
и в `--ff-display` поставьте первым `'Azo Sans'`.

## Платёжные кнопки

Все CTA размечены как `<a href="#" data-product="..." data-price="...">`.
Список:

| data-product       | где                          | действие                                |
|--------------------|------------------------------|-----------------------------------------|
| `hero-cta`         | hero «Занять место»          | скролл к `#tariffs`                     |
| `guarantee-cta`    | блок гарантии                | скролл к `#tariffs`                     |
| `tariff-probyu`    | тариф ПРОБУЮ ($35)           | вставить ссылку оплаты                  |
| `tariff-delayu`    | тариф ДЕЛАЮ ($69)            | вставить ссылку оплаты                  |
| `tariff-prodayu`   | тариф ПРОДАЮ ($150)          | вставить ссылку оплаты                  |

Для подключения оплаты замените `href="#"` на реальный URL (WayForPay/Stripe/CryptoCloud/другое):

```html
<a href="https://secure.wayforpay.com/payment/abcd1234"
   class="btn btn--pink"
   data-product="tariff-prodayu"
   data-price="150">Купить</a>
```

Или замените логику в `scripts.js` (блок `// -------- Кнопки покупки`)
на свою — например, отправку формы или открытие модалки оплаты.

## Контент-плейсхолдеры

Все картинки сейчас представлены как `<div>` с классом-плейсхолдером
(градиентный фон). Они помечены `data-asset="..."` и `aria-label`-ом.
Чтобы вставить настоящие изображения:

1. Положите файл в `assets/` (например `assets/step-1-table.png`).
2. Замените соответствующий `<div data-asset="step-1-table">` на:

```html
<img src="./assets/step-1-table.png" alt="..."
     class="img-placeholder img-placeholder--shadow"/>
```

Список плейсхолдеров с `data-asset`:

```
hero               — превью видео в hero (9:16)
step-1-table       — скрин таблицы аккаунтов
step-2-views       — скрин таблицы просмотров
step-3-reels-1..3  — три обложки залетевших Reels
stats-main         — основной скрин статистики
stats-side         — скрин с приростом подписчиков
phone-1, phone-2   — мокапы телефонов с примерами
result-1a..5b      — обложки видео в блоке «Результат»
program-m1..m4-video — превью видео для каждого модуля
```

## Программа практикума (табы)

Реализована на нативном JS + ARIA. По нажатию `←/→` фокус и активная
вкладка переключаются. Аккордеоны уроков — нативный `<details>/<summary>`,
работает без JS.

Если нужно «открыто всегда» — уберите атрибут `hidden` со всех `data-panel`
и скройте `.program-tabs__nav`.

## Производительность

- Шрифты подгружаются через Google Fonts с `preconnect`.
- Нет внешних JS-библиотек, полный JS-бандл < 2 КБ.
- Все интерактивные блоки прогрессивно улучшаются: без JS табы превратятся
  в видимый только первый модуль (можно поправить, удалив `hidden`).

6. **OG-теги и фавикон**: дополнить в `<head>` (сейчас базовые).
7. **Sitemap.xml / robots.txt** при необходимости.
